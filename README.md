## ✨ Overview / Resumen

**Hoard** is a professional-grade, privacy-first digital vault. Version 3.0 introduces powerful new ways to collect media, especially images, while keeping everything organized and private on your machine.

**Hoard** es una bóveda digital profesional con la privacidad como prioridad. La versión 3.0 introduce nuevas y potentes formas de coleccionar medios, especialmente imágenes, manteniendo todo organizado y privado en tu propia máquina.

---

## 🚀 Key Features / Características Principales

### 🌐 Advanced Browser Extension / Extensión Avanzada
- **⚡ Massive Image Import**: One-click import of entire boards or galleries from sites like **Pinterest, Flickr, Unsplash, Behance, and Dribbble**.
- **📂 Smart Organization**: Automatically detects board/album names to create and organize items into folders.
- **📥 Local Downloads**: Not just links — Hoard can download and store images locally for offline access.
- **📦 Importación Masiva**: Importa tableros completos o galerías de sitios como **Pinterest, Flickr, Unsplash, Behance y Dribbble** con un solo clic.
- **📂 Organización Inteligente**: Detecta automáticamente nombres de tableros o álbumes para crear y organizar elementos en carpetas con nombre.
- **📥 Descarga Local**: No solo guarda enlaces; Hoard puede descargar y almacenar las imágenes localmente para acceso offline.

### 📂 Organization & Power Tools / Organización y Herramientas
- **📂 Multi-Vault & Folders**: Nested organization with named folders for high-density collections.
- **🔍 Power Search**: Global search and Command Palette (`Cmd/Ctrl + K`) to find anything in milliseconds.
- **📝 Rich Notes**: Full Markdown support powered by Tiptap.
- **💻 Code Snippets**: Syntax highlighting for dozens of languages.
- **🖼️ OCR Integration**: Extract text from your saved images automatically.

## 🛠️ Tech Stack / Tecnologías

- **Frontend**: [React](https://reactjs.org/) + [Vite](https://vitejs.dev/)
- **Desktop Framework**: [Electron](https://www.electronjs.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) + [Radix UI](https://www.radix-ui.com/)
- **Database**: [SQLite (sql.js)](https://sql.js.org/)
- **OCR Engine**: [Tesseract.js](https://tesseract.projectnaptha.com/)

## 📦 Installation / Instalación

### Developer Setup
```bash
git clone https://github.com/XsharklinX/Hoard.git
cd Hoard
npm install
npm run dev
```

### Browser Extension
1. Go to `chrome://extensions/`.
2. Enable **Developer mode**.
3. Click **Load unpacked** and select the `extension` folder.

---

## 🔒 Privacy / Privacidad

Hoard is built on the philosophy that your data should be yours.
- **Zero Cloud**: No data is sent to external servers.
- **Local Database**: All information is stored in a local SQLite file.
