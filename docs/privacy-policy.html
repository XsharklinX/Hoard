Save to Hoard Privacy Policy

Last updated: 2026-05-30

Save to Hoard helps users save and organize web content locally in their browser and, when available, sync that content to the Hoard Desktop app running on the same computer.

Data Processed

The extension processes only content that the user explicitly chooses to save, including URLs, page titles, selected text, image URLs, notes, code snippets, folder selections, and tag selections.

Storage and Transfer

Saved content is stored locally in the browser. When Hoard Desktop is running, the extension may send saved content to the local Hoard Desktop service at 127.0.0.1. Save to Hoard does not send saved content to a developer-operated cloud service or sell user data.

Permissions

The extension uses browser permissions to save selected content, provide context-menu actions, store offline captures, retry pending captures, search local items, scan galleries only when requested, and display the saved-items side panel.

User Control

Users can delete locally saved browser items from the side panel and export locally stored browser data from the popup.
