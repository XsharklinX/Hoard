Features

Everything you need.
Nothing you don't.

Hoard packs the features of Obsidian, Raindrop, and Pocket — into a single local app, with no subscription.

🗂️

6 Types of Content

Links with metadata, rich text notes, images with OCR, code with syntax highlight, quotes with attribution, and file attachments — all in one place.

🔗 Link 📝 Note 🖼 Image 💻 Code ❝ Quote 📎 File
🔒

100% Local

Everything is stored in a SQLite database on your machine. No cloud sync, no telemetry, no account. Your data never leaves your computer.

🔍

Powerful Search

Full-text search with SQLite FTS5. Use operators like before:, tag:, has:archive, lang:python and save frequent searches.

🕸️

Knowledge Graph

Connect notes with [[wiki links]], see backlinks and related items, explore your knowledge as an interactive d3-force graph.

📡

RSS Feed Reader

Subscribe to RSS/Atom feeds, import from OPML, and new articles land directly in your vault — automatically, in the background.

✏️

Rich Editor

TipTap v3 with slash commands, callout blocks, KaTeX math, Mermaid diagrams, tables, task lists, and wiki links.

📤

Export Anywhere

Export as Markdown, PDF, or a complete navigable static HTML site. Share individual items with a hoard:// URL that anyone can import.

🗄️

Multi-vault

Keep separate vaults for work, personal, research. Smart folders with AND/OR conditions auto-organize your items.

Advanced Search

Find anything, instantly

Type natural search operators right in the search bar. Filter by date, type, status, language, or content — and save your most-used searches in the sidebar.

before:2024-01-01 Items created before a date
unread:yes Only unread items
has:archive Items with saved page archive
lang:python Code in a specific language
dead:yes Broken links to clean up
tag:research domain:github.com Combine operators
🔍 local-first type:link unread:yes | Save ⌘S

Browser Extension

Save from anywhere.
Even without the app.

The Hoard extension works independently. Save to your local browser collection and sync to the desktop app whenever you want.

🌐

Works offline

Saves to IndexedDB in your browser. No desktop app required.

❝

Smart selection

Select text → choose Note, Quote, or Copy as Markdown.

🖼

Board import

Import all images from Pinterest, Flickr, Unsplash with one click.

📖

Offline reader

Article text is saved automatically. Read later, even when offline.

🔄

Auto-sync

Items sync to the desktop app in the background when it's running.

How it works

Up and running in minutes

1

Download & install

Download the installer for your platform and run it. Hoard starts in your system tray.

→
2

Create a vault

A vault is your personal collection. Create one on first launch — it's a simple SQLite file on your drive.

3

Start saving

Install the browser extension and use Alt+S on any page. Or use the app directly.
