# Save to Hoard Privacy Policy

Last updated: 2026-05-30

Save to Hoard helps users save and organize web content locally in their
browser and, when available, sync that content to the Hoard Desktop app running
on the same computer.

## Data Processed

The extension processes only content that the user explicitly chooses to save,
including URLs, page titles, selected text, image URLs, notes, code snippets,
folder selections, and tag selections.

## Storage and Transfer

Saved content is stored locally in the browser. When Hoard Desktop is running,
the extension may send saved content to the local Hoard Desktop service at
`127.0.0.1`. Save to Hoard does not send saved content to a developer-operated
cloud service or sell user data.

## Permissions

- `activeTab`: reads the current page title and URL when the user saves a page.
- `contextMenus`: provides right-click save actions.
- `storage` and `unlimitedStorage`: stores offline captures locally.
- `alarms`: retries pending local captures while Hoard Desktop is available.
- `omnibox`: searches saved local items from the browser address bar.
- `scripting`: scans a page for images only after the user starts a gallery
  import.
- `sidePanel`: displays saved local items in the browser side panel.

## User Control

Users can delete locally saved browser items from the side panel and export
their locally stored browser data from the popup.
